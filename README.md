# ZentraPOS Documentation Site

This project is a static documentation website built for GitHub Pages.

## Files
- `index.html` – the full documentation site
- `styles.css` – site styling
- `script.js` – sidebar, search, active section highlighting, and image lightbox
- `assets/images/` – screenshots used in the manual

## Deploy to GitHub Pages
1. Create a new GitHub repository.
2. Upload all project files in this folder.
3. Commit and push to the `main` branch.
4. In GitHub: **Settings → Pages**
5. Under **Build and deployment**, choose:
   - **Source:** Deploy from a branch
   - **Branch:** `main` and `/root`
6. Save.

GitHub will publish the site at:

`https://YOUR-USERNAME.github.io/YOUR-REPOSITORY/`

## Editing content
- Replace screenshots inside `assets/images/`
- Edit section content directly in `index.html`
- Update colors or layout in `styles.css`

## Notes
This site is static HTML/CSS/JS. It does not need a backend or build step.
